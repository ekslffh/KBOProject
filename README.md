### 소개
• 구단별로 다양하게 존재하는 야구예매 사이트를 전체 통합으로 관리하기  
• 모든 야구 팬들을 한데 모아서 커뮤니티도 형성  
• 구단별 상품도 한곳에서 구매가 가능하도록 구현
